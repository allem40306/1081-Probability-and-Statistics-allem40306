# 題目說明
* DateCheck：日期
* IntCheck、IntCheckSum：二、八、十六進位整數
* IPcheck：IPv4
* TimeCheck：時間
* p1：擷取所有字串，依字典序大小輸出
* p2：判斷四個用 `-` 分隔的字串是否為整數，如果皆為整數輸出總和
* p3：碼掉電話號碼
* p4：判斷是否整個數字為浮點數，如果不是，把符合的部分輸出
